Installing on Windows
===
0. Uninstall any previous EMS Cart driver
1. Connect the EMS 64M cart to the USB port
2. Let Windows fail to find any driver for it
3. Open zadig_2.2.exe
4. Make sure that the selected driver is WinUSB and the USB ID is 4670 9394
   and WCID is *not* selected
5. Press "Install driver"
6. Open ems-qart.exe
7. Enjoy!

To report a bug, please open an issue on Github (https://github.com/rbino/ems-qart)
